# Crypton
Crypton est open source ! Créez votre serveur !
Comment le lancer :

## Cloner le git
`git clone https://github.com/ChatHivernal/Crypton.git`

## Entrer dans le dossier
`cd Crypton`

## Installer les dépendences
`pip3 install -r requierements.txt`

## Executer le script
`python3 app.py`

## Allez sur le site
Dans le navigateur `localhost:3030`
